**Author**: Brian Kimbrow  
**Version**: 1.0.0 (increment the patch/fix version number if you make more commits past your first submission)

## Overview

The City Explorer project enables users to search for cities they want to visit or learn more about. When users search for a city, the application provides the city's longitude, latitude, and current weather. This is a full-stack project created using Vite React,  and Express.

## Getting Started

1. Clone the repository to your local machine.
2. Navigate to the directory where you cloned the repo.
3. Run `npm install` to install project dependencies.
4. Execute `npm run dev` to start the development server.


## Architecture

This project uses Vite React for the frontend, and Express for the backend API.
